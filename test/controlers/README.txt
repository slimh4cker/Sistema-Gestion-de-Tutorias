Esta carpeta se utiliza para realizar pruebas sobre los controler directamete
De esta forma se salta el paso de ingresar con una cuenta
Estas pruebas NO funcionan para probar las apis y los middlewares asosiados a estos
Para ello utilize las APIS dentro de la carpeta ENDPOINTS

Los metodos de aqui no requieren que el servidor este andando

Utiliza libreria JEST, para instalarlo utiliza en la terminal
npm insstall --save-dev jest

para correr todos los test utiliza el comando:
npm run test

Eso es si dentro del package.json esta este script incluido
